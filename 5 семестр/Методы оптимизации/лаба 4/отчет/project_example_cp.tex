\item[\sffamily 1.]При помощи Квазиньютоновских методов с точностью $\varepsilon=10^{-3}$ найти точку минимума квадратичной функции
\[f(x,y)=11x^2+3y^2+6xy-2\sqrt{10}(x-3y)-22.\]
Начальное приближение $X^0=(\sqrt{10},0)^T,X^0=(0,\sqrt{10})^T$.\\
Ответ:$(1.58,-4,47)^T$.\\\\
При исследовании квадратичной квадратичной функции с различными начальным приближением все алгоритмы сработали идентично:\\
