Разумов Т.Е., Швечков И.В.
\end{center}
Пусть для функции $\sqrt{x+1}-1=0$ корень локализован на отрезке $[-1,1]$. С использованием аналитической производной метод сошелся к корню $x^*=0$ за 4 итерации. При использовании численной производной при $\varepsilon=10^{-15}$, метод сошелся к корню за 7 итераций.
